# 💪 Maui-AppFit

A cross-platform mobile application for fitness and workout tracking, built with .NET MAUI. Helps users monitor their exercises and stay active.

## 📱 About The Project

Maui-AppFit is a simple and practical tool for fitness enthusiasts who want to track their workouts. The app allows users to log exercises and monitor their progress.

This project was created for study purposes and to demonstrate the capabilities of .NET MAUI for cross-platform mobile and desktop development.

- Track workouts and exercises
